Eric Patrizio

Artisan logiciel, influenceur Agile
Software craftsman, agile enthusiast

Paris, France.

Je suis développeur backend (environ 20 ans d'expérience en développement logiciel dans des environnements web) spécialisé en PHP, Symfony, PostgreSQL. Je porte une grande attention à la qualité du code source, je m'intéresse particulièrement à l'approche DDD (Domain Driven Design) dans l'écriture de mon code métier. Je suis avec attention les mouvements "Artisanat du Logiciel" et "DevOps".
De manière générale, je suis passionné par les valeurs portées par l'Agilité, les communautés Open Source et le logiciel libre. Depuis 2018, je suis membre de l'équipe d'organisation de la conférence Agile France.
